## 自己管理サイトの強み（重要）

業者に制作を依頼したサイトは、業者が廃業・倒産した場合に以下のリスクがある：
- ドメイン・サーバーを業者が管理していた場合 → **サイトごと消える**
- ソースコードを渡してもらえていない場合 → **手元に何も残らない**
- 独自CMSの場合 → **ログインすらできなくなる**
- 次の業者への依頼で数十万円の再制作費が発生する

**西笑院サイトはこのリスクが一切ない：**
- ファイルはすべて自分のGitHub（Microsoft運営）に保管
- 業者・制作者に依存していない
- GitHubが存続する限りサイトは永続する
- Claude Codeを解約してもサイト・ファイルは消えない

> 斟珠寺（shinsyuji.jp）も同様にClaude Codeで制作されており、
> 業者制作サイト（50〜100万円相当）と同等のクオリティを

### 方法① Claude Codeに頼む（一番かんたん）
「v1に戻して」と言うだけ。自動でやってくれます。

### 方法② GitHub Desktop を使う（Claude Codeなしでもできる）

1. **GitHub Desktop を開く**
2. 上部メニュー「**History**」タブをクリック
3. 左側の履歴一覧から **「マップ埋め込みをGoogleマップ公式URLに変更」** を探す
   （これがv1.0 = 2026年5月23日の安定版）
4. そのコミットを**右クリック** →「**Revert Changes in Commit（このコミットを元に戻す）**」
5. 「**Push origin**」ボタンを押す → サイトが元に戻る

### 方法③ GitHub のウェブサイトから（パソコン不要・スマホでも可）

1. ブラウザで https://github.com/seisyoinn1560-web/tosotsu を開く
2. 右上の「**Code**」→「**index.html**」をクリック
3. 右上の時計アイコン「**History**」をクリック
4. 「マップ埋め込みをGoogleマップ公式URLに変更」のコミットを選択
5. 右上「**<> Browse files**」→「**index.html**」→「**Raw**」
6. 全文をコピーして、手元の index.html に貼り付けて保存

### 方法④ フォルダ内のバックアップファイルを使う（最もかんたん）

1. フォルダ内の **`index_v1_backup.html`** を開く
2. ファイル名を **`index.html`** にコピー（上書き）
3. GitHub Desktop で **Commit → Push** するだけ
