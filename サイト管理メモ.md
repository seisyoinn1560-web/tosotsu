## サイトを更新する方法

サイトの内容を変更したいときは以下の手順です。

### 1. ファイルを編集する
- `index.html` を編集（Claude Codeで作業）

### 2. GitHubに送る（これだけで自動公開される）
以下のコマンドをターミナルで実行：

```
cd /mnt/d/西笑院ホームページサイト
git add .
git commit -m "更新内容のメモ"
git push origin main
```

### 3. 自動で公開される
GitHubに送ると Netlify が自動でサイトを更新します。1〜2分で反映されます。
